Welcome to ChatScript Help

Thank you for choosing our chat script! Here's a guide to help you get started:

Registration:

  1. Create an Account:
    • To use the chat, every user needs to register.
    • Go to register and enter required information to set up your username and password.

Login to Sessions:

  1. Login to a Session:
    • Once registered, log in using the login button.
    • You can either select an existing session or create a new one from the dropdown menu.
    • Each session will have a unique date name.
    • If you want to join an existing session, select the existing session from the dropdown menu.
    • To create a new session, select the create new session and enter your details.

Session Access:

  • Each session can be accessed by any registered user.
  • If you received a link with a session, just enter your account details to join.

Quick Access:

  1. Share Session Links:
    • Make chatting easier by sharing session links.
    • You can share the session by sending the link of the session to your friends, but they need to be registered.
